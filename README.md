# Bangladesh Premire Leauge Auction 2021
## Player Auction Site.
### Live site link is https://bpl-auction2020.netlify.app/

<h3> 👨🏻‍💻 &nbsp;Features </h3>
- 🤔 &nbsp; Dynamic Route
- 🎓 &nbsp; Fake data has been used for data
- 💼 &nbsp; You can select player from availabe option.
- 💼 &nbsp; Total salary are computed according to their salary 
- 💼 &nbsp; Total number of selected player can also be shown besides salary.
